# Clash Verge 分流脚本说明

本项目提供适用于 **Clash Verge Rev / Mihomo Party** 的分流覆写脚本，用于实现更清晰、易维护的网络策略分流结构。

脚本采用多订阅合并 + 分组策略路由方式，支持自动加入订阅节点，并按功能分类流量。

---

## 🧩 策略组说明

| 策略组 | 作用说明 |
|------|--------|
| 默认直连 | 默认直连出口（中国大陆直连流量） |
| 默认拒绝 | 直接丢弃流量（广告/非法请求） |
| 节点选择 | 总代理节点选择器（含订阅自带节点） |
| 国外媒体 | YouTube / Netflix / Spotify / TikTok 等海外媒体 |
| 国内媒体 | Bilibili / 爱奇艺 / 腾讯视频 / 网易云等 |
| 国内社交 | 微信 / QQ / 微博 / 小红书等国内社交 |
| AIGPT | OpenAI / Claude / Gemini / AI 服务 |
| Steam | Steam 下载与社区流量 |
| Telegram | Telegram 消息与 CDN 流量 |
| 广告拦截 | 广告 / 追踪 / 恶意域名屏蔽 |
| 漏网之鱼 | 未匹配到规则的兜底流量 |
